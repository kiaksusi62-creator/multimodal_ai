# 🌐 Multimodale KI – alle kostenlosen KI-Modelle per API

Diese App vereint über 20 kostenlose KI-Modelle in einer einzigen Chat-Oberfläche.  
Sie nutzt einen intelligenten Router, der automatisch das beste Modell für Text, Code, Bildgenerierung, Sprachsynthese, Spracherkennung, PDF-Analyse und Video-Fragen auswählt.
